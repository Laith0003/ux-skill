Thesis

The output has fingerprints.

Default model output reaches for the same patterns every time. We catalogued them — every gradient, every filler verb, every centered hero. Every command in this plugin reads the bans before it ships a pixel.

Pure black #000000. The purple-to-blue AI gradient on white. 3 equal cards in a row. John Doe, Acme, $99.99. A centered hero over a dark image. Decorative animation with no meaning. Filler verbs — Elevate, Seamless, Unleash.

What ships

Curated, not generated.

A reference corpus the model can’t bypass. Read on every command call. Versioned in git. Open to PRs.

30
cognitive UX laws codified

Hick, Fitts, Jakob, Miller, von Restorff — named, indexed, with concrete rules attached.

Each law lives in a reference file. When a command needs to reason about choice overload or scan-paths, it cites the law by name and applies the constraint. No reinvention, no vibes.

21 premium SaaS sites pulled apart pattern by pattern.

Linear, Vercel, Stripe, Raycast, Arc, Pitch, Cron, Mercury and 13 more — not as inspiration boards, but as a documented arsenal of patterns with their use cases and performance costs.

audit lenses, one report

Layout. Type. Color. Content. Motion. Components.

Each lens has its own forbidden-patterns catalogue. The audit runs all six in sequence and emits a single report with severity tags — Critical, High, Medium, Low — so the fix order is obvious.

How it fits

Three minutes from clone to first audit.

A symlink, not a package manager. Update by pulling the repo. Uninstall by removing the link.

  1. 01

    Clone the repo.

    Public, MIT, no auth required. Anywhere on disk you keep checkouts. 

    $ git clone https://github.com/Laith0003/ux-skill.git
  2. 02

    Step into it.

    The next command runs from inside the repo so the symlink resolves the absolute path correctly. 

    $ cd ux-skill
  3. 03

    Symlink into Claude Code.

    Claude Code looks under ~/.claude/plugins/ at startup. One symlink, one restart, plugin loaded. 

    $ ln -s "$(pwd)" ~/.claude/plugins/ux
  4. 04

    Run your first audit.

    Restart Claude Code, open a project, run any slash command. Output starts citing the bans by name. 

    > /ux-audit ./app/page.tsx

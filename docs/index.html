the discipline

Default AI output
has fingerprints.
We catalogued them.

Pure black #000000. The purple-to-blue AI gradient on white. Three equal cards in a row. "John Doe", "Acme", 99.99%. Centered hero over a dark image. Decorative animation with no meaning. Filler verbs — "Elevate", "Seamless", "Unleash". A 3-line testimonial with no name attached. The unconstrained generator reaches for these reflexively. The result reads as machine-made within seconds.

8 of 200+ catalogued. Every command reads the list before producing anything — refusing the safest default is half the discipline.

the surface area

grouped by intent.

before shipping it.

a real run

The audit names
the violation,
names the fix.

Every finding has a severity tag, an evidence line, and a fix. The verdict at the bottom is a number, not vibes.

Output sample below — actual command, actual page, slightly tightened for the page width.

~/audit-report 3 critical 
$ /ux-audit https://my-app.com/pricing

─── findings ──────────────────────────
[CRITICAL]  [SCAN]   Krug: Omit Needless Words
            Evidence: hero h1 is 6 lines (147 words)
            Fix:      Tighten to 2 lines, ≤22 words

[HIGH]      [ACT]    Hick's Law
            Evidence: 11 plan tiers on the pricing grid
            Fix:      Collapse to 3 tiers + "Compare all"

[MEDIUM]    [READ]   error specificity
            Evidence: "Form contains errors"
            Fix:      "Email is required. Add an email and try again."

─── verdict ────────────────────────────
3 critical · 7 high · 12 medium · ship-readiness: NO-GO
Recommended next: /ux-copy --fix (highest leverage)

install

Install in 60 seconds.

Clone the repo, symlink into ~/.claude/plugins/, restart Claude Code. All 17 commands appear in /help.

Get the plugin

the four steps

Four lines between you
and a working session.

  1. 01

    Clone the repository

    Public on GitHub, MIT licensed. No login, no waitlist.

    terminal
    $ git clone https://github.com/Laith0003/ux-skill.git
  2. 02

    Symlink into Claude Code

    One line. The plugin lives where Claude Code expects it.

    terminal
    $ ln -s "$(pwd)/ux-skill" ~/.claude/plugins/ux
  3. 03

    Restart Claude Code

    The plugin loader picks up the new directory at boot.

    terminal
    $ claude restart
→ ux plugin loaded · 17 commands · 5 sub-agents
  4. 04

    Run /ux-frame

    Frame the brief before building. Every other command compounds from here.

    claude code
    > /ux-frame "landing for a niche developer tool"
→ asking 10 questions to nail audience, voice, style, wow…

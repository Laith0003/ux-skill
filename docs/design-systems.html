Make your own in one command

Every system here was generated, not curated. Point the engine at a brief and it synthesizes a fresh, accessible design language and writes the same folder:

ux system-pack --industry fintech-payments --tone bold --name "Cobalt"

Deterministic, offline, free. See the commands, read what a DESIGN.md is, or grab the machine-readable index.

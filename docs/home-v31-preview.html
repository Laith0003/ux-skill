01 — How it works

Brief in. Brand-true page out. No model in the loop.

Five deterministic steps. The same brief always compiles the same system — and you can read every decision.

uxskill discover
# 10-field intake, improvisation banned ✓ brief locked: industry, brand, references, wow # the wow is named as ONE concrete moment
uxskill recommend
# 5 parallel lanes → one system style Saturated Cinema · palette multi-hue type Bricolage / Inter / JetBrains Mono ✓ anchored to your brand primary
uxskill synthesize
# 7-axis design language, compiled warmth .40 contrast 1.0 density .70 motion .75 type_personality .80
uxskill lint --threshold high
# 152 deterministic regex rules, no LLM scanned 1 file · 0 high · sub-50ms # AI-slop caught before commit
node verify-responsive.mjs 360,390
# real headless Chrome, true viewports 360px scrollWidth=360 ok · sticky 60px 390px scrollWidth=390 ok ✓ VERIFIED clean · exit 0

03 — New in 3.1

Brand-true. And it survives a phone.

ANCHORED

The primary comes from your logo's pixels.

Default fonts are rejected for your logo's letterform. A brand-fidelity hard floor fails any output that drops your colour, logo or real imagery — no house-style drift.

GATED

Mobile verified for real, not asserted.

A headless render at true 360/390 viewports fails horizontal scroll, a wrapping nav or wordmark, and an over-tall sticky header. A gate that can't render says so — never a false green.

DERIVED

It reaches for a wow moment.

The engine derives a 2–3-moment wow layer from the brief instead of shipping flat. The page you're reading is the dogfood.

Stop shipping the centroid.
